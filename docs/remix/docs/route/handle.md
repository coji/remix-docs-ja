# `handle`

ハンドルをエクスポートすると、[`useMatches`][use-matches]フックを使用してアプリケーションの慣例を作成できます。ハンドルには、任意の値を置くことができます。

```tsx
export const handle = {
  its: "all yours",
};
```

これは、ほとんどの場合、`useMatches`と組み合わせて使用されます。ハンドルでできることについては、詳細を学ぶために[`useMatches`][use-matches]を参照してください。
