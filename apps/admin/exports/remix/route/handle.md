# `handle`

`handle` をエクスポートすることで、[`useMatches`][use-matches] フックを使ってアプリケーションの規約を作成できます。任意の値を設定できます。

```tsx
export const handle = {
  its: "all yours",
};
```

これはほとんどの場合、`useMatches` と組み合わせて使用されます。`handle` で何ができるかについては、[`useMatches`][use-matches] を参照してください。
