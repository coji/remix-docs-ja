# redirect

[MODES: framework, data]

## 概要

[リファレンスドキュメント ↗](https://api.reactrouter.com/v7/functions/react_router.redirect.html)

リダイレクトレスポンス。ステータスコードと `Location` ヘッダーを設定します。デフォルトは "302 Found" です。

## シグネチャ

```tsx
redirect(url, init): Response
```
